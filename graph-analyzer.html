Graphical Network Analyzer — Help

Input Format

Paste one directed edge per line as two node names separated by a space or tab:

WTP       V-E1-300
V-E1-300  V-CDFG-500
V-E1-300  V-AB-350
WTP       WTP

Each line becomes a directed edge from the left node to the right node. Lines with more or fewer than two tokens are flagged as Malformed. A node pointing to itself is flagged as a Self-Loop and skipped from the graph.

Preprocessing Modes

  • Do Not Alter Input Names — node names used exactly as typed.
  • Ignore Symbols Only — strips non-alphanumeric characters, uppercases.
  • Ignore Prefix & Symbols — additionally strips any leading letters, leaving only the numeric part.
  • Add Lat-Long to Nodes — reveals a table of all nodes parsed from the input. Enter lat and lng per node, then click Apply & Render in the coordinate panel. Nodes with coordinates use GPS layout; others fall back to auto-layout.

Analysis Panels

  • Repeated / Bidirectional — A→B appears more than once (Repeated), or both A→B and B→A exist (Bidirectional).
  • Triangles — three mutually connected nodes: A–B, B–C, A–C all present.
  • Self-Loops — a node listed as its own neighbour (A→A).
  • Nodes with 4+ Branches — nodes with four or more connections; highlighted red in the graph.
  • Malformed Lines — lines that did not parse as exactly two tokens.

Click any highlighted item in the sidebar to zoom the graph to that node or group.

  ],
  "edges": [
    {
      "data": {
        "id": "WTP_V-E1",
        "source": "WTP",
        "target": "V-E1"

  ]
